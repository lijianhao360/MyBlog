1-注释

  • 注释是在程序指定位置添加的说明性信息
  • 简单理解，就是对代码的一种解释

注释内容不会参与编译和运行,仅仅是对代码的解释说明

  • 如class(用于创建/定义一个类,类是Java最基本的组成单元),

4-变量

变量就是在程序执行的过程中有可能发生改变的量(数据)


变量注意事项:
1.变量只能存一个值
2.变量名不允许重复定义
3.一条语句可以定义多个变量

4.变量在使用前一定要进行赋值

day-02

1-基本数据类型

基本数据类型概述:
Java的数据类型分为两大核心类别：基本数据类型（8种）和引用数据类型（如字符串、数组、类等）。

基本数据类型是Java的基础数据单元，直接存储数据值，特点是占用空间固定、存取速度快。

基本数据类型分类（8种）：

  • 数值型:整数型 byte short int long（4种）
  • 浮点型: float double（2种）
  • 字符型：char（1种）
  • 布尔型：boolean（1种）

整数型(4种):
用于存储整数（正数、负数、0），不同类型占用内存空间不同，取值范围也不同：

类型 占用空间 取值范围 适用场景
byte 1字节 -128 ~ 127 节省内存（如文件流、网络传输）
short 2字节 -32768 ~ 32767 较少使用，兼容老旧系统
int（默认） 4字节 -2¹⁵ ~ 2¹⁵-1（约±21亿） 日常整数使用（如计数、编号）
long 8字节 -2⁶³ ~ 2⁶³-1 存储大数（如时间戳、金额）

2-标识符

标识符的命名规则

  • 由数字,字母,下划线(_)和美元$组成
  • 不能以数字开头
  • 不能是关键字
  • 区分大小写

软性建议
小驼峰命名法：方法,变量
规范1: 标识符是一个单词的时候全部小写
eg1: name

规范2: 标识符由多个单词组成的时候，第一个单词首字母小写，其他单词首字母大写
eg2: firstName

大驼峰命名法： 类名
规范1: 标识符是一个单词时，首字母大写
eg1: Student

规范2: 标识符是由多个单词组成的时候，每个单词首字母大写
eg2: GoodStudent
